A high-performance **Analytics Dashboard** built with React, TypeScript, MUI, and ECharts.  
Designed to visualize data interactively with dynamic charts, filters, and smooth UI interactions.  
Fully responsive, production-ready, and deployed on **Render** (Frontend + Backend).

---

## 🚀 Live Demo  
🔗 **Frontend:** *https://analytics-uqhv.onrender.com*  
🔗 **Backend API:** *https://analytics-80al.onrender.com*

---

## ⭐ Key Features  
### **Frontend**
- 📈 Interactive charts powered by **ECharts**  
- 🎨 Clean UI built with **MUI**  
- 🌗 Light/Dark theme toggle  
- 🔍 Dynamic data filters  
- 📱 Fully responsive for mobile and desktop  
- ⚡ Fast rendering with optimized React components  
- 🔁 Real-time data fetching from backend API  

### **Backend**
- 🔧 Built using **Python + Flask**  
- 📡 REST APIs for analytics data  
- 🔐 CORS-enabled secure communication  
- ⚙️ Configurable endpoints for future scalability  
- 🚀 Deployed on Render with auto-build  

## 🛠️ Technologies Used
